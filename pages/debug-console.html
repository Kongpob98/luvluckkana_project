🔍 Gemini API Debug Console

📋 วิธีดูข้อมูลที่ส่งให้ Gemini API

1️⃣
เปิด Chatbot และ Developer Console
  • เปิด chatbot.html
  • กด F12 หรือ Cmd+Option+I (Mac)
  • ไปที่แท็บ Console
2️⃣
ส่งข้อความในแชทบอท
ตัวอย่าง: "ทำนายดวงราศีมังกร"
3️⃣
ดู Debug Logs ใน Console
คุณจะเห็น: 
🔍 DEBUG: Gemini API Request
  📝 System Prompt Length: 1234 characters
  💬 User Message: ทำนายดวงราศีมังกร
  📚 Knowledge Retrieved: 567 characters
  📤 Full Prompt: [ข้อความทั้งหมดที่ส่งไป]

🔍 DEBUG: Full Request Body
  {
    "contents": [...],
    "generationConfig": {...},
    "safetySettings": [...]
  }

📥 Response status: 200

🔍 DEBUG: Full Response
  Response Data: {...}
  📊 Token Usage:
    promptTokens: 450
    completionTokens: 150
    totalTokens: 600

🌐 Network Tab

ขั้นตอน:
  • F12 → แท็บ Network
  • กรอง: generativelanguage.googleapis.com
  • คลิกที่ request ชื่อ generateContent
  • ดูแท็บ:
    • Headers - ข้อมูล Request
    • Payload - ข้อมูลที่ส่งไป
    • Response - ข้อมูลที่ได้รับกลับ

🏠 Google AI Studio

ประวัติการใช้งาน:
  • เปิด AI Studio History
  • ดูทุก request ที่เคยส่ง
  • ดู prompt และ response
  • ตรวจสอบการใช้ quota

🎯 ข้อมูลที่คุณจะเห็น

✅ Request Data (ข้อมูลที่ส่งไป):
  • System Prompt: คำสั่งให้ AI ทำหน้าที่เป็นโหร (~1,500 ตัวอักษร)
  • Knowledge Base: ข้อมูลราศี/ดวง/เลขศาสตร์ที่เกี่ยวข้อง
  • User Message: คำถามของคุณ
  • Generation Config: temperature, topK, topP, maxOutputTokens
  • Safety Settings: การตั้งค่าความปลอดภัย
📊 Response Data (ข้อมูลที่ได้กลับ):
  • Text: คำตอบจาก AI
  • Prompt Tokens: จำนวนโทเค็นที่ส่งไป
  • Completion Tokens: จำนวนโทเค็นที่ตอบกลับ
  • Total Tokens: รวมทั้งหมด
  • Safety Ratings: คะแนนความปลอดภัย

⚙️ การตั้งค่า Debug Mode

⚠️ ตอนนี้ DEBUG_MODE = true ใน chatbot.js 
// ใน js/components/chatbot.js บรรทัดที่ 10
const DEBUG_MODE = true;  // เปิด Debug
// const DEBUG_MODE = false; // ปิด Debug (ใช้ตอน production)
คำแนะนำ:
  • ✅ เปิด DEBUG_MODE = true ตอนพัฒนา
  • ⚠️ ปิด DEBUG_MODE = false ก่อน publish
  • 📊 Debug จะแสดง console.log ทุกขั้นตอน

💡 Tips

  • 🔍 ใช้ Console Filter: พิมพ์ 🔍 เพื่อดูแค่ Debug logs
  • 📋 Copy Request: คลิกขวา → Copy object เพื่อ copy ข้อมูล
  • 💾 Save Log: คลิกขวาใน Console → Save as...
  • 🎨 Theme: Console รองรับ Dark/Light mode
  • ⚡ Performance: ดูเวลาตอบกลับใน Network timing tab
